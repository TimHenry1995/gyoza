import os, sys

  sys.path.insert(0, x[0])
update version in conf.py of docs
ensure conf.py has theme 'sphinx_rtd_theme', ensure extennsions = ['sphinx.ext.todo','sphinx.ext.viewcode','sphinx.ext.autodoc']

cd to docs directory, run make html
